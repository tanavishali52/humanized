# SEO Audit Skill

Use this skill to ensure every page is optimized for search engines.

## SEO Checklist

1.  **Metadata**: Does the page have a descriptive `title` and `meta description`?
2.  **Headings**: Is there exactly one `<h1>` per page? Is the heading hierarchy logical (h1 > h2 > h3)?
3.  **Semantic HTML**: Are semantic tags like `<main>`, `<section>`, `<article>`, and `<nav>` used appropriately?
4.  **Image Alt Text**: Do all images have descriptive `alt` attributes?

## Feedback Format

- List missing or weak SEO elements.
- Suggest specific text for titles and meta descriptions.
- Recommend structural changes for better crawlability.
